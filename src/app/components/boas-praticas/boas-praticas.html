Chegando

Boas práticas com Angular

Um curso de Angular do setup ao deploy, com foco em arquitetura, performance, acessibilidade e produtividade.

Cadastrados na lista recebem condições especiais de lançamento.

Pra quem é este curso

  • Devs que já conhecem o básico de Angular e querem profissionalizar
  • Devs vindos de outras versões que querem se atualizar pro Angular moderno
  • Quem trabalha em projetos corporativos e quer padrões escaláveis
  • Quem quer fundamentar decisões técnicas com critérios, não com achismo

Pré-requisitos: TypeScript intermediário e familiaridade básica com Angular (componentes, templates, services).

O que você vai aprender

Módulo 1: Projeto moderno do zero
  • Criando o projeto com Angular CLI (zoneless, Vitest, SSR, esbuild+Vite)
  • Estrutura de pastas por features: core, features, shared
  • ESLint com angular-eslint e regras estritas
  • Prettier e consistência de código
  • Configuração do TypeScript (strict mode, paths)
  • Convenções de nomenclatura
  • Processo de atualização do Angular e dependências
Módulo 2: Decisões de arquitetura no dia a dia
  • Template-driven vs reactive forms: critérios objetivos pra escolher
  • Content projection vs inputs: quando cada padrão escala melhor
  • Componentes inteligentes vs componentes de apresentação
  • Organização de serviços: quando um service vira god class
  • Interceptors em cadeia: ordem, efeitos colaterais e boas práticas
  • Injeção de dependência avançada: InjectionToken, hierarquia de injetores
  • Memory leaks, circular dependencies e barrel files
  • Padrões de comunicação entre componentes
Módulo 3: Reatividade com Signals
  • Fundamentos: signal(), computed(), effect()
  • Effects: o que fazer e o que evitar
  • Interop com RxJS: toSignal(), toObservable()
  • Resource API para acesso a dados
  • Signal inputs, two-way binding e model signals
  • Signal Forms: validators, schemas, nested forms, custom controls
  • Estado local vs global: quando signals bastam
Módulo 4: Estado e gerenciamento de dados
  • Arquitetura de estado: local, feature-level, global
  • NgRx SignalStore: criando stores leves
  • View models e estado normalizado
  • Gerenciamento de entidades com withEntities()
  • Custom features para requisitos recorrentes
  • Side effects baseados em RxJS
  • SignalStore + Resource API + httpResource
Módulo 5: Performance
  • Ferramentas: Lighthouse, Web Vitals, bundle analyzer
  • NgOptimizedImage e otimização de imagens
  • Bundle size, tree shaking e impacto de libs de terceiros
  • Lazy loading e estratégias de preloading
  • @defer: deferrable components e libraries
  • SSR, hybrid rendering e prerendering (SSG)
  • Hydration, partial hydration e event replay
  • Change detection: OnPush, signals e zone-less
  • Virtual scrolling com CDK
Módulo 6: Acessibilidade
  • Ferramentas: Lighthouse a11y, Wave, leitores de tela
  • WAI, WCAG 2.2, ARIA roles e atributos
  • HTML semântico e estrutura de documento
  • Acessibilidade no Angular Router
  • Angular Material e CDK para acessibilidade
  • CSS acessível: foco visível, contraste, prefers-reduced-motion
Bônus: Produtividade com IA
  • Usando IA para pesquisa e aprendizado Angular
  • Geração de código: modelos, componentes, testes
  • GitHub Copilot e copilot-instructions.md
  • Angular MCP Server

Perguntas frequentes

Preciso saber Angular pra fazer o curso?

Um básico sobre componentes, templates, services e roteamento é bem-vindo. O curso é focado em boas práticas e padrões de arquitetura, então não é um curso de introdução ao Angular. Se você é iniciante, recomendo começar pelo meu livro.

Qual versão do Angular?

O curso será gravado com a versão mais recente disponível no momento do lançamento.

Em que formato são as aulas?

Videoaulas gravadas que você assiste no seu ritmo, construindo uma aplicação real do zero.

Tem certificado?

Sim, ao concluir todos os módulos você recebe um certificado de conclusão.

E se eu tiver dúvidas durante o curso?

Detalhes sobre o suporte serão divulgados junto com o lançamento.
